10 etiquetas de HTML

1. HTML Video Element

El atributo de controles agrega controles de video, como reproducción, pausa y volumen. Es una buena idea incluir siempre atributos de ancho y alto. Si no se configuran el alto y el ancho, la página puede parpadear mientras se carga el video. El elemento < source > le permite especificar archivos de vídeo alternativos entre los que el navegador puede elegir. El navegador utilizará el primer formato reconocido. El texto entre las etiquetas < video > y < /video > solo se mostrará en navegadores que no admitan el elemento < video >.


2. Youtube Video

Para poder reproducir un video de YouTube en su pagina web, debe seguir los siguientes paso:

  • Subir el video a Youtube
  • Conseguir el ID del video
  • Definir un elemento < iFrame > en la pagina web
  • Dejar que el atributo scr apunta al video
  • Usar las funciones width and height para especificar las dimensiones del reproductor


3. iFrame Target de Enlace

Un iframe HTML se utiliza para mostrar una página web dentro de una página web. Se puede utilizar un iframe como marco de destino para un enlace. El atributo de destino del enlace debe hacer referencia al atributo de nombre del iframe:

9. HTML < blockquote > para Citar

El elemento HTML < blockquote > define una sección citada de otra fuente. Los navegadores suelen aplicar sangría a los elementos < blockquote >.
